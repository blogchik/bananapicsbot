Hozirgi trend promptlar/style’larni 1-2 bosishda ishlatish.

## Manba

MVP: admin curated ro‘yxat (`trending_items`):

- yoki prompt override

## Qo‘shimcha (keyinroq)

- Auto-trending (analytics asosida)
- Time window (starts_at/ends_at)
- Region/language bo‘yicha variantlar

## UX
